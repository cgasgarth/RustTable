# RustTable release notes

RustTable is in active migration from darktable to a native Rust application.
No stable RustTable release is published yet, and this document must not claim
darktable versions, compatibility, camera support, checksums, or packages as
RustTable releases.

## Unreleased

### Migration status

- The active product is the Rust workspace in `crates/`, with GTK4 through
  gtk-rs as its UI framework.
- The original darktable checkout remains a separate, read-only reference for
  source-accounted behavior and parity evidence.
- Public release artifacts are deferred until the required package, platform,
  security, and validation issues have accepted evidence.

### Release-entry rules

Add an entry only when the merged RustTable change has user-visible behavior
or materially changes supported platforms, packages, security posture, or
migration compatibility. Each entry should identify the RustTable issue/PR and
describe the verified behavior without implying unimplemented darktable
coverage.

Use these headings when the first release is prepared:

- Added
- Changed
- Fixed
- Security
- Packaging and platform support
- Known limitations

Every published release will include artifact names, immutable checksums,
supported-platform scope, upgrade notes, and reproducible validation evidence
generated by RustTable's release workflow.
